La diferencia:

include<file>: Se utiliza para inicializar archivos o paqueterias de sistema, se busca un archivo de nombre "file" en la lista de directorios.


inclued "file": Usado para encabezados de archivo de programacion local.


Static storage: La instruccion le dice al compilador que mantenga la existencia local de una variable mientras el programa se ejecuta. Declarar de manera local las variables del estilo static permite mantener sus valores entre funciones.
