1.-¿Investiga lo que quieren decir las letras EOF. ¿Por qué se debe definir c como int?

EOF significan End Of File. La letra c se define como int para habilitar a la variable EOF.

2.- Modifica main para eliminar la línea: c = getchar(); antes del while y hacer el llamado a getchar en la condición del while.

#include <stdio.h>
main(void){
    int c;
    while( (c = getchar()) != EOF){
        putchar(c);
    } 
}


3.-Utiliza getchar para escribir un programa que cuente el número de caracteres dado un stdin e imprima este número. Al dar enter (salto de línea) el programa termina. Por ejemplo:
$echo -e "hola\n" | ./ejecutable.out
entonces el programa imprime 4

$echo -e "hola\n" | ./tarea_2acountchar.out

#include <stdio.h>
#include <string.h>

int main(void){
    char str[100];
    int c, n = 0, len;
    
    while( (c = getchar()) != '\n'){
        str[n] = c;
        n++;
        
    } 
    len = strlen(str);
    printf("%d\n", len);
    printf("\n");

return(0);
}
