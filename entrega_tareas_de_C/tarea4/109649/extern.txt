La declaración de variables mediante extern le hará saber al compilador sobre el uso de variables declaradas en 
el entorno global para ser usadas, manipuladas o procesadas en entornos locales, las cuales conservarán los 
valores modificados que se les haya otorgado o modificado a lo largo de cada proceso local.

extern hace que pueda importarse el valor anterior de una variable global a un proceso local.
