3) Utiliza getchar para escribir un programa que cuente el nmero de caracteres dado un stdin e imprima este nmero. Al dar enter (salto de lnea) el programa termina.
#include <stdio.h>
main(void){
    int c;
    int i =0;
    while( (c = getchar()) != '\n'){
        i++;
    }
    printf("%d\n",i);
}
